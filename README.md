## üíª Local development
### Setup
```
$ git clone git@github.com:zentala/id.zentala.io.git
$ cd id.zentala.io.git
$ nvm use .
$ npm install
```

### Development mode
```
$ npm run dev
```
Then open in the browser `http://localhost:2000/`.

### Build and deploy
```
$ npm run deploy
```
